# NYC Inputs API

> Backend API for NYC Inputs application, which is a survey website

## Usage

Rename "config/config.env.env" to "config/config.env" and update the value/settings to your own

## Install Dependencies

```
npm install
```

# Run app
```
# Run in dev mode
npm run dev

# Run in prod mode
npm start
```
